Portfolio

  • HTML / CSS / JavaScript
  • PHP
  • Heroku

HTML,CSSべた書きで作ったサイトに記事管理機能やメールフォームを付けた自分用のポートフォリオサイトです。
HTML5から導入されたHistoryAPIのpushState()メソッドを利用して、SPA（シングルページアプリケーション）を構築しました。

記事ページのデザインはQiitaやGitHubのREADME等を参考にしました。 記事ページは4つの方法で外部にシェア可能です。
