# Portfolio
HTML,CSSべた書きで作ったサイトに記事管理機能やメールフォームを付けた自分用のポートフォリオサイトです。  
HistoryAPIのpushState()メソッドを利用して、SPA（シングルページアプリケーション）を構築しています。
