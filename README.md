# Portfolio
HTML,CSSべた書きで作ったサイトにPHPで記事管理機能やメールフォームを付けた自分用のポートフォリオサイトです。  
HTML5から導入されたのHistoryAPIのpushState()メソッドを利用して「ページ遷移しないWebシステム」を導入しています。

- 完成サイト  
https://aokiyuki-portfolio.herokuapp.com/
